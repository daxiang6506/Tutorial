## 快捷键

  ```
  Ctrl + Alt + -    back一个浏览位置(linux)
  Ctrl + Shift + -  forward一个浏览位置(linux)
  Ctrl + -          back一个浏览位置(mac)
  Ctrl + Shift + -  forward一个浏览位置(mac)
  F12               跳转到定义
  Ctrl + Tab        切换选项卡
  Ctrl + p          找文件
  ```
