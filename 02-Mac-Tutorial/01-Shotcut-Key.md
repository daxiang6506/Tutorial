## 快捷键
* Command + Shift + .
  >显示隐藏文件/文件夹
* Command + Shift + G
  >finder下前往任何文件夹，包括隐藏文件夹
* Option + space
  >打开Alfred，这个需要自己安装Alfred，并设置快捷键
* Option + Command + C
  >复制文件路径
* Command + 向上
  >上一级目录
* Command + Shift + x
  >打开当前文件下的终端
